# kastl

> A work-in-progress terminal client for Home Assistant.

`kastl` currently focuses on a CLI workflow and is intended to grow into a TUI over time.

## Requirements

- Rust and Cargo
- A reachable Home Assistant instance
- A Home Assistant long-lived access token

## Run

```bash
cargo run -- --help
```

## Notes

- Early-stage project
- Commands and interfaces are still changing
